# מערכת מעקב תלמידים

פרויקט זה מציג מערכת לניהול משתמשים ומעקב מיקומים של תלמידים ומורים על גבי מפה.
המערכת כוללת מסך התחברות, מסך הרשמה, ומסך פנימי המציג את נתוני המשתמשים והמיקומים הקיימים.

## אופן השימוש באתר

### מסך התחברות

בתמונה הראשונה ניתן לראות את מסך ההתחברות.
במסך זה ניתן להיכנס עם שם משתמש קיים באמצעות הזנת `ID` ו-`Password`.
הכניסה למערכת הפנימית מורשית רק למורות.

![מסך התחברות](imgs/image%20(3).png)

### מסך הרשמה

בתמונה השנייה ניתן לראות את מסך ההרשמה.
במסך זה ניתן להירשם לאתר בתור סטודנט או מורה, על ידי הזנת תעודת זהות, שם מלא, כיתה, תפקיד וסיסמה.

![מסך הרשמה](imgs/image%20(4).png)

### מסך המערכת הפנימי

בתמונה השלישית ניתן לראות את תצוגת האתר מבפנים עם דוגמה למשתמשים ומיקומים קיימים.
בנוסף, ניתן לסנן את הנתונים לפי מרחק, או לפי אחת מהאפשרויות הבאות:

- `My class` - רק התלמידים שלי
- `All students` - כל התלמידים
- `All teachers` - כל המורים

![מסך פנימי](imgs/image%20(5).png)

## התקנה והרמת השירותים

המערכת מורכבת משני שירותים עיקריים:

- שרת `FastAPI`
- מסד נתונים `MongoDB`

### דרישות מקדימות

- `Python 3.10` ומעלה
- `Docker`
- `Docker Compose`

### שלב 1: התקנת התלויות

יש ליצור סביבה וירטואלית ולהתקין את התלויות הדרושות:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install fastapi uvicorn "beanie<2" motor
```

### שלב 2: הרמת מסד הנתונים

יש להרים את שירות MongoDB באמצעות:

```bash
docker compose up -d
```

פקודה זו מעלה את מסד הנתונים לפי ההגדרות שבקובץ `docker-compose.yml`.

### שלב 3: טעינת נתוני דוגמה

כדי לטעון משתמשים ומיקומים התחלתיים למערכת יש להריץ:

```bash
python seed.py
```

הסקריפט טוען נתוני דוגמה של מורים, תלמידים ומיקומים להצגה במפה.

### שלב 4: הרמת שרת האפליקציה

לאחר מכן יש להפעיל את שרת האפליקציה:

```bash
python main.py
```

לאחר שהשרת עולה, ניתן לגשת למערכת בכתובת:

[http://127.0.0.1:8000/app/](http://127.0.0.1:8000/app/)

### שלב 5: עצירת השירותים

כדי לעצור את מסד הנתונים יש להריץ:

```bash
docker compose down
```

## סיכום

המערכת מאפשרת:

- כניסה עם משתמש קיים
- הרשמה בתור סטודנט או מורה
- צפייה במשתמשים ובמיקומים קיימים על גבי מפה
- סינון נתונים לפי מרחק או לפי סוג המשתמשים המוצגים
